## Screenshot Rules

- Use only fictional demo data.
- Do not show real names, real employers, real contact details, real resumes, or private user feedback.
- Avoid showing local file paths if possible.
- Crop screenshots so the main feature is easy to understand.
- Use clear filenames with lowercase letters and hyphens.
